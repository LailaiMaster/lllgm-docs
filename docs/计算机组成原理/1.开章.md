# CPU的性能
主要有两个指标

# 响应时间或执行时间

让计算机跑的更快。

越小越好。

# 吞吐率或带宽

让计算机干的更多

性能=1/响应时间 

# 计算机的计时单位：CPU 时钟

程序的 CPU 执行时间 =CPU 时钟周期数×时钟周期时间

时钟周期是什么，可以粗浅的任务，CPU的主频2.8Hz，1秒执行2.8G条指令。

程序的 CPU 执行时间 = 指令数×CPI（**每条指令的平均时钟周期数**）×Clock Cycle Time
